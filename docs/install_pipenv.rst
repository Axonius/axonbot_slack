Using pipenv
#####################################################

Using `pipenv <https://github.com/pypa/pipenv>`_ allows you to install this into a virtual environment.

Install pipenv
=====================================================
Install pipenv first:

.. code-block:: console

   $ pip install --upgrade pipenv
