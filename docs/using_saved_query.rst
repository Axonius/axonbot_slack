Saved Query commands
#####################################################
Commands to work with Saved Queries.

Saved Query list commands
====================================================

List Saved Queries for devices
----------------------------------------------------
* :blue:`saved query devices`: Reply to you in a thread with a list of all valid Saved Queries for devices.

  .. image:: _static/images/axonbot_saved_query_devices.png
     :scale: 60

List Saved Queries for users
----------------------------------------------------
* :blue:`saved query users`: Reply to you in a thread with a list of all valid Saved Queries for users.

  .. image:: _static/images/axonbot_saved_query_users.png
     :scale: 60

Get Saved Query for devices
----------------------------------------------------
* :blue:`saved query devices [VALUE]`: Reply to you in a thread with the objects from the Saved Query supplied as ``VALUE`` in JSON format.

  .. image:: _static/images/axonbot_saved_query_devices_query.png
     :scale: 60
