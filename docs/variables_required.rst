Required Variables
#####################################################

SLACK_API_TOKEN
=====================================================
API Token of bot user in Slack.

See :ref:`Steps to generate an API token for a Slack bot`.

AX_URL
=====================================================
URL of Axonius instance to connect to.

See :ref:`Steps to get an Axonius API Key and API Secret`.

AX_KEY
=====================================================
API Key of Axonius user at AX_URL instance.

See :ref:`Steps to get an Axonius API Key and API Secret`.

AX_SECRET
=====================================================
API Secret of Axonius user at AX_URL instance.

See :ref:`Steps to get an Axonius API Key and API Secret`.
