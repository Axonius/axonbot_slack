Prerequisites
#####################################################

Connection to the Slack API
=====================================================
The URL used to access the Slack API is: https://slack.com

The system you run this bot on needs to be able to access the Slack API outbound only.

If you need a proxy to get out to the internet, you can use :ref:`HTTPS_PROXY`.

Connection to the Axonius instance
=====================================================
The system you run this bot on needs to be able to access the Axonius instance outbound only.

If you need a proxy to get to the Axonius instance, you can use :ref:`AX_HTTPS_PROXY`.
